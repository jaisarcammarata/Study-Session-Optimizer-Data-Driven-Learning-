## 📌 About the Project:
This project was born from a personal question: **Is there a specific time of day or energy level that maximizes my focus and retention?** 

As a Data Analysis learner and automation enthusiast, I decided to stop guessing and start measuring. This repository contains the ecosystem to track, process, and eventually predict my optimal study sessions based on real-world data.

## 🚀 Project Phases:
1.  **Data Capture:** A Python script for manual session logging (Current Phase 📍).
2.  **Automation Pipeline:** Structured storage in CSV/SQL format.
3.  **Exploratory Data Analysis (EDA):** Identifying patterns in fatigue and productivity using Pandas and Seaborn.
4.  **Predictive Modeling:** Implementing a basic Machine Learning model to suggest study durations.

## 📁 Repository Structure:
* `src/`: Core scripts (Logger, Data Processing).
* `data/`: Storage for session logs (CSV).
* `notebooks/`: Detailed analysis and data visualizations (Coming soon).
